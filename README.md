# saurian-translator

A fun encoding/decoding project that translates between English and a fictional language called Saurian.
Demonstrates string manipulation, loops, and character-shift logic.

## Features
- Translate predetermined English → Saurian
- Translate predetermined Saurian → English
- Handles full sentences in the test cases
- Produces consistent, reversible translations
- Includes automated test cases
  
## How to Run
- Runs on:
- OnlineGDB
- Replit
- Any local Java compiler/IDE

## Compile
javac Saurian.java SaurianDriver.java

## Run
java SaurianDriver
